TuiImage Struct Reference

Image loaded from a file or generated through code. This object contains an image texture, which can be loaded into a TuiGlyphAtlas. TuiImage objects are passable between multiple threads and are independent of TuiInstance objects. This object is opaque, and its members are not meant to be accessed directly. Use the functions in image.h instead.
