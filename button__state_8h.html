◆ TuiButtonState

enum TuiButtonState

The position of a button or key.

Enumerator
TUI_BUTTON_INVALID 

The button state is invalid.

This button state is returned when an error occurs, and is also used to specify that no button state is defined.

TUI_BUTTON_RELEASE 

The button is released.

TUI_BUTTON_PRESS 

The button is pressed.

TUI_BUTTON_REPEAT 

The button is pressed and is repeating.

TUI_BUTTON_FIRST 

The first valid button state.

TUI_BUTTON_LAST 

The last valid button state.
