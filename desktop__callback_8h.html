◆ tuiFileDropFunction

typedef void(* tuiFileDropFunction) (TuiWindow window, int path_count, const char *paths[])

Callback type used for TuiWindow file drop events.

Parameters
windowThe TuiWindow that captured the event.
path_countThe amount of files/folders that were dropped in the window.
pathsAn array of the full paths of the dropped files.
Pointer lifetime:
paths is only safe to access until a file drop callback is called again, at which point it will be overwritten.
