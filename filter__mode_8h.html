◆ TuiFilterMode

enum TuiFilterMode

The filter mode used for rendering a texture or framebuffer.

Enumerator
TUI_FILTER_INVALID 

The filter mode is invalid.

This filter mode is returned when an error occurs, and is also used to specify that no filter mode is defined.

TUI_FILTER_POINT 

Filter mode with no filtering.

TUI_FILTER_BILINEAR 

Filter mode with bilinear color interpolation.
