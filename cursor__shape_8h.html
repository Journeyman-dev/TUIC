◆ TuiCursorShape

enum TuiCursorShape

Default mouse cursor shapes.

Enumerator
TUI_CURSOR_SHAPE_INVALID 

The cursor shape is invalid.

This cursor shape is returned when an error occurs, and is also used to specify that no cursor shape is defined.

TUI_CURSOR_SHAPE_ARROW 

The default arrow cursor shape.

TUI_CURSOR_SHAPE_IBEAM 

The cursor shape used for selecting or inserting text.

TUI_CURSOR_SHAPE_CROSSHAIR 

A plus sign shaped cursor.

TUI_CURSOR_SHAPE_POINTING_HAND 

A pointing hand cursor, normally used when hovering over hyperlinks or buttons.

TUI_CURSOR_SHAPE_RESIZE_EW 

The east-west resize cursor.

TUI_CURSOR_SHAPE_RESIZE_NS 

The north-south resize cursor.

TUI_CURSOR_SHAPE_RESIZE_NWSE 

The northwest-southeast resize cursor.

TUI_CURSOR_SHAPE_RESIZE_NESW 

The northeast-southwest resize cursor.

TUI_CURSOR_SHAPE_RESIZE_ALL 

The resize cursor for all directions.

TUI_CURSOR_SHAPE_NOT_ALLOWED 

The cursor for not allowed actions.

TUI_CURSOR_SHAPE_HRESIZE 

The east-west resize cursor.

TUI_CURSOR_SHAPE_VRESIZE 

The north-south resize cursor.

TUI_CURSOR_SHAPE_FIRST 

The first valid cursor shape.

TUI_CURSOR_SHAPE_LAST 

The last valid cursor shape.
