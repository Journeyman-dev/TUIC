◆ TuiKeyboardMod

enum TuiKeyboardMod

Keyboard modifier keys.

Enumerator
TUI_MOD_INVALID 

The keyboard mod is invalid.

This keyboard mod is returned when an error occurs, and is also used to specify that no keyboard mod is defined.

TUI_MOD_SHIFT 

A shift key is pressed.

TUI_MOD_CONTROL 

A control key is pressed.

TUI_MOD_ALT 

An alt key is pressed.

TUI_MOD_SUPER 

A super key is pressed.

TUI_MOD_CAPS_LOCK 

A caps lock key is pressed.

TUI_MOD_NUM_LOCK 

A num lock key is pressed.

TUI_MOD_NONE 

Now mod keys are pressed.

TUI_MOD_ALL 

All mod keys pressed.
