TuiPalette tuiPaletteCreateColors ( int  channel_count,
int  color_count,
const uint8_t *  color_data 
)

Create a custom TuiPalette with a colors array.

Parameters
channel_countThe amount of channels per color in the array.
color_countThe amount of colors in the array.
color_dataA pointer to the colors array.
Errors:
Possible errors in order are TUI_ERROR_NOT_INITIALIZED, TUI_ERROR_INVALID_PALETTE_COLOR_COUNT, TUI_ERROR_INVALID_CHANNEL_COUNT, and TUI_ERROR_NULL_COLORS. The first error that occurs will cause the function to immediatly return. Also, an inccorectly sized or colors array may cause undefined behaviour or a fatal crash without an error.
Requirements:
This function must be called only while TUIC is initialized.
Pointer lifetime:
The returned TuiPalette must be destroyed before TUIC is terminated, using the function tuiPaletteDestroy().
Thread safety:
This function must only be called on the same thread on which TUIC was initialized to ensure safe memory access and to prevent graphics context errors.
