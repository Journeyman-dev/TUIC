Create a new TuiAtlas of type TUI_ATLAS_GRID with a texture from a codepage TuiImage.

Parameters
imageThe TuiImage to use as the texture of the created atlas.
Returns
The created TuiAtlas. TUI_NULL is returned if an error occurs.
Errors:
Possible errors in order are TUI_ERROR_NOT_INITIALIZED, TUI_ERROR_NULL_IMAGE, TUI_ERROR_INVALID_CODEPAGE_DIMENSIONS, and TUI_ERROR_INVALID_BLEND_MODE. The first error that occurs will cause the function to immediatly return.
Requirements:
This function must be called only while TUIC is initialized.
Pointer lifetime:
The returned TuiAtlas must be destroyed before TUIC is terminated, using the function tuiAtlasDestroy().
Thread safety:
This function must only be called on the same thread on which TUIC was initialized to ensure safe memory access and to prevent graphics context errors.

◆ tuiGenerateUVCoordinatesFromPixelCooordinates()

float* tuiGenerateUVCoordinatesFromPixelCooordinates ( int  glyph_count,
uint16_t *  pixel_coordinates,
int  texture_width,
int  texture_height,
float *  out_ptr 
)

Get a raw UV texture coordinate STPQ array from a raw pixel coordinate STPQ array.

Parameters
glyph_countThe amount of glyphs.
pixel_coordinatesThe pixel coordinate array.
texture_widthThe width of the texture in pixels.
texture_heightThe height of the texture in pixels.
out_ptrA pointer to a correctly sized array where the output will be stored. If TUI_NULL, a new array is allocated on the heap instead.
Returns
The UV texture coordinate array.
Errors:
This function can have no errors. However, a malformed pixel coordinate array or a misized out_ptr array may cause undefined behaviour or a fatal crash without an error.
Requirements:
This function can be called freely, even if TUIC is not currently initialized.
Pointer lifetime:
The returned UV texture coordinate array must be freed with tuiFree() when it is no longer needed to avoid memory leaks.
Thread safety:
This function does not access memory in ways that could cause a data race, so it is thread safe.
