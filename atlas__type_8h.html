enum TuiAtlasType

How the glyph locations within a TuiAtlas are stored.

Enumerator
TUI_ATLAS_INVALID 

The atlas type is invalid.

This atlas type is returned when an error occurs, and is also used to specify that no atlas type is defined.

TUI_ATLAS_COORDS 

The atlas uses an STPQ texture coordinate array to store the locations of each glyph.

The STPQ coordinates for each glyph are in order of glyph index, with the first four elemeents in the array being the STPQ coordinates of the glyph of index 0.

TUI_ATLAS_GRID 

The glyphs are layed out in a grid pattern where each glyph has the same pixel width and pixel height.

Glyphs are indexed from left to right, top to bottom, with the index of 0 being the glyph in the upper right corner of the texture.

TUI_ATLAS_FIRST 

The first valid atlas type value.

TUI_ATLAS_LAST 

The last valid atlas type value.
