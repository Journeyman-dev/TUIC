◆ tuiPaletteCreate()

TuiPalette tuiPaletteCreate ( TuiInstance  instance,
int  channel_count,
int  color_count,
uint8_t *  color_data 
)

Use this function to create a TuiPalette using an array of colors. The array must be in order of the color indices in the pallete, in order of their RGB components.

Parameters
instanceThe TuiInstance object that contains the graphics context.
channel_countThe amount of color channels per color.
color_countThe amount of colors.
color_dataA pointer to an array that contains all of the colors.
Returns
The created TuiPalette object. NULLL is returned on error.
Errors
Throws TUI_ERROR_NULL_INSTANCE if instance is NULL. Throws TUI_ERROR_DAMAGED_INSTANCE if instance is damaged. Throws TUI_ERROR_INVALID_PALETTE_COLOR_COUNT if color count is not greater than 0 and less than or equal to 256. Throws TUI_ERROR_INVALID_CHANNEL_COUNT if channel_count is not 3 or 4. Throws TUI_ERROR_NULL_COLORS if color_data is NULL. Throws TUI_ERROR_BACKEND_SPECIFIC and may or may not return if backend specific errors occur.
