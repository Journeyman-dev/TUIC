◆ tuiPanelGetPixels()

uint8_t* tuiPanelGetPixels ( TuiPanel  panel,
int *  pixel_width,
int *  pixel_height,
uint8_t *  fill_pixels 
)

Get the raw pixels of the framebuffer of a TuiPanel.

Parameters
panelThe TuiPanel to get the pixels from.
pixel_widthA pointer to where the pixel width of the panel will be stored. If TUI_NULL or an error occurs, it is ignored.
pixel_heightA pointer to where the pixel height of the panel will be stored. If TUI_NULL or an error occurs, it is ignored.
fill_pixelsA pointer to an array store the pixels array in. If the sizes specified in pixel_width and pixel_height do not fit the pixels, the array is reallocated. If TUI_NULL, a new array is allocated instead.
Returns
The pixels array.
Errors:
Possible errors in order are TUI_ERROR_NOT_INITIALIZED and TUI_ERROR_NULL_PANEL. The first error that occurs will cause the function to immediatly return.
Requirements:
This function must be called only while TUIC is initialized.
Pointer lifetime:
If fill_pixels is used, it must point to an array created with tuiAllocate(). The returned pixels array must be destroyed with tuiFree().
Thread safety:
This function must only be called on the same thread on which TUIC was initialized to ensure safe memory access and to prevent graphics context errors..
