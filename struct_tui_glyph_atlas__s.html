TuiGlyphAtlas Struct Reference

TUI glyph image atlas. This object contains a image for rendering glyphs as well as definitions of where glyphs are in the image. These objects should only be used on the same thread as its TuiInstance. This object is opaque, and its members are not meant to be accessed directly unless implementing a backend. Use the functions in glyph_atlas.h instead.
