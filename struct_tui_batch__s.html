TuiBatch Struct Reference

TUI rendering data container, which contains data about tiles to draw. TuiBatch objects are passable between multiple threads and are independent of TuiInstance objects. This object is opaque, and its members are not meant to be accessed directly. Use the functions in batch.h instead.
