enum TuiDetailFlag

Bitflag components that are combined into TuiDetailMode enums.

Enumerator
TUI_DETAIL_FLAG_INVALID 

The detail flag is invalid.

This detail flag is returned when an error occurs, and is also used to specify that no detail flag is defined.

TUI_DETAIL_FLAG_GLYPH_8 

TUI detail mode glyph flag for 8 bit glyphs. This supports up to 256 unique glyphs. This size is optimal if you only want to use code page 437 for rendering.

TUI_DETAIL_FLAG_GLYPH_16 

Detail mode glyph flag for 16 bit glyphs. This supports up to 65536 unique glyphs per TuiAtlas.

TUI_DETAIL_FLAG_COLOR_0 

TUI detail mode flag for no colors. This shrinks the byte size per tile by not requiring any information about foreground and background colors.

TUI_DETAIL_FLAG_COLOR_4 

TUI detail mode color flag for 4 bit colors. This enables use of a 16 color TuiPalette for colors.

TUI_DETAIL_FLAG_COLOR_8 

TUI detail mode color flag for 8 bit colors. This enables use of a 256 color TuiPalette for colors.

TUI_DETAIL_FLAG_COLOR_8NFG 

TUI detail mode color flag for 8 bit colors with no foreground color. This enables use of a 256 color TuiPalette for colors.

TUI_DETAIL_FLAG_COLOR_8NBG 

TUI detail mode color flag for 8 bit colors with no background color. This enables use of a 256 color TuiPalette for colors.

TUI_DETAIL_FLAG_COLOR_24 

TUI detail mode color flag for 24 bit colors. This enables use of 24 bit full colors with RGB components.

TUI_DETAIL_FLAG_COLOR_24NFG 

TUI detail mode color flag for a single 24 bit background color per tile. This enables use of 24 bit full colors with RGB components.

TUI_DETAIL_FLAG_COLOR_24NBG 

TUI detail mode color flag for a single 24 bit foreground color per tile. This enables use of 24 bit full colors with RGB components.

TUI_DETAIL_FLAG_COLOR_32 

TUI detail mode color flag for 32 bit colors. This enables use of 32 bit full colors with RGBA components.

TUI_DETAIL_FLAG_COLOR_32NFG 

TUI detail mode color flag for a single 32 bit background color per tile. This enables use of 32 bit full colors with RGBA components.

TUI_DETAIL_FLAG_COLOR_32NBG 

TUI detail mode color flag for a single 32 bit foreground color per tile. This enables use of 32 bit full colors with RGBA components.

TUI_DETAIL_FLAG_LAYOUT_FULL 

TUI detail mode color flag for full batches of tiles without increased batch size for coordinate information. This is most optimal for rendering in situations where almost all tiles are updated every frame.

TUI_DETAIL_FLAG_LAYOUT_SPARSE 

TUI detail mode color flag for rendering tiles as needed instead of every tile every frame. This is most optimal for rendering in situations where few tiles are updated every frame.
