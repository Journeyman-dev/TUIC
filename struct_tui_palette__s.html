TuiPalette Struct Reference

TUI color palette. This object contains an array of colors that can be used for the drawing of batch data arrays with a TuiDetailMode that requires a palette. These objects should only be used on the same thread as its TuiInstance. This object is opaque, and its members are not meant to be accessed directly unless implementing a backend. Use the functions in palette.h instead.
