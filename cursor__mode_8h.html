◆ TuiCursorMode

enum TuiCursorMode

The state of the mouse cursor.

Enumerator
TUI_CURSOR_MODE_INVALID 

The cursor mode is invalid.

This cursor mode is returned when an error occurs, and is also used to specify that no cursor mode is defined.

TUI_CURSOR_MODE_NORMAL 

The cursor is visilble and behaves normally.

TUI_CURSOR_MODE_HIDDEN 

The cursor is hidden while over the window and behaves normally outside the window.

TUI_CURSOR_MODE_DISABLED 

The cursor is hidden and unable to be removed from the window.

TUI_CURSOR_MODE_FIRST 

The first valid cursor mode.

TUI_CURSOR_MODE_LAST 

The last valid cursor mode.
