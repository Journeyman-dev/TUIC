enum TuiErrorCode

Error codes for errors caused by the TUIC library.

Enumerator
TUI_ERROR_UNKNOWN 

TUIC error code for when an unknown error has occured.

TUI_ERROR_NONE 

TUIC error code for when no error has occured.

TUI_ERROR_ALREADY_INITIALIZED 

TUIC error code for when TUIC has already been initialized.

TUI_ERROR_GRAPHICS_BACKEND_SPECIFIC 

TUIC error code for when a graphics backend specific error has occured.

TUI_ERROR_DANGLING_ATLAS 

TUIC error code for when attempting to terminate TUIC with one or more dangling TuiAtlas.

TUI_ERROR_DANGLING_CURSOR 

TUIC error code for when attempting to terminate TUIC with one or more dangling TuiCursor.

TUI_ERROR_DANGLING_PALETTE 

TUIC error code for when attempting to terminate TUIC with one or more dangling TuiPalette.

TUI_ERROR_DANGLING_PANEL 

TUIC error code for when attempting to terminate TUIC with one or more dangling TuiPanel.

TUI_ERROR_DANGLING_TEXTURE 

TUIC error code for when attempting to terminate TUIC with one or more dangling TuiTexture.

TUI_ERROR_DANGLING_WINDOW 

TUIC error code for when attempting to terminate TUIC with one or more dangling TuiWindow.

TUI_ERROR_GLFW_API_UNAVAILABLE 

TUIC error code for when a GLFW error has occured: GLFW could not find support for the requested API on the system.

TUI_ERROR_GLFW_FORMAT_UNAVAILABLE 

TUIC error code for when a GLFW error has occured: the requested format is not supported or available.

TUI_ERROR_GLFW_INVALID_ENUM 

TUIC error code for when a GLFW error has occured: one of the arguments to the function was an invalid enum value.

TUI_ERROR_GLFW_INVALID_VALUE 

TUIC error code for when a GLFW error has occured: one of the arguments to the function was an invalid value.

TUI_ERROR_GLFW_NOT_INITIALIZED 

TUIC error code for when a GLFW error has occured: GLFW has not been initialized.

TUI_ERROR_GLFW_NO_CURRENT_CONTEXT 

TUIC error code for when a GLFW error has occured: No context is current for this thread.

TUI_ERROR_GLFW_NO_WINDOW_CONTEXT 

TUIC error code for when a GLFW error has occured: The specified window does not have an OpenGL or OpenGL ES context.

TUI_ERROR_GLFW_OUT_OF_MEMORY 

TUIC error code for when a GLFW error has occured: a memory allocation failed.

TUI_ERROR_GLFW_PLATFORM_ERROR 

TUIC error code for when a GLFW error has occured: a platform-specific error occurred that does not match any of the more specific categories.

TUI_ERROR_GLFW_VERSION_UNAVAILABLE 

TUIC error code for when a GLFW error has occured: the requested OpenGL or OpenGL ES version is not available.

TUI_ERROR_INCOMPATIBLE_BLEND_MODE 

TUIC error code for when the TuiBlendMode of the TuiAtlas is incompatible with the TuiDetailMode of the batch data or TuiBatch.

TUI_ERROR_INCOMPATIBLE_IMAGES 

TUIC error code for when images have incompatible channel counts.

TUI_ERROR_INVALID_ASPECT_RATIO 

TUIC error code for when an aspect ratio is numerator or denominator is less than or equal to 0.

TUI_ERROR_INVALID_ATLAS_TYPE 

TUIC error code for when a TuiAtlasType is invalid.

TUI_ERROR_INVALID_BATCH_DATA_DIMENSIONS 

TUIC error code for when the tile dimensions of a batch data are less than or equal to 0.

TUI_ERROR_INVALID_BATCH_DIMENSIONS 

TUIC error code for when the tile dimensions of a TuiBatch are less than or equal to 0.

TUI_ERROR_INVALID_BATCH_SETTER 

TUIC error code for when a batch tile setter was used for the incorrrect TuiDetailMode.

TUI_ERROR_INVALID_BLEND_MODE 

TUIC error code for when a TuiBlendMode is invalid.

TUI_ERROR_INVALID_BUTTON_STATE 

TUIC error code for when a TuiButtonState is invalid.

TUI_ERROR_INVALID_CHANNEL_COUNT 

TUIC error code for when a channel count is invalid.

TUI_ERROR_INVALID_CODEPAGE_DIMENSIONS 

TUIC error code for when the pixel dimensions of codepage TuiAtlas is not divisible by 16 and greater than 0.

TUI_ERROR_INVALID_CURSOR_MODE 

TUIC error code for when a TuiCursorMode is invalid.
