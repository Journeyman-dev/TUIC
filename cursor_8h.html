int  hotspot_x,
int  hotspot_y 
)

Create a new TuiCursor from a 4 channel TuiImage.

Parameters
imageThe TuiImage to use to create the cursor image.
hotspot_xThe x pixel coordinate of the cursor hotspot.
hotspot_yThe y pixel coordinate of the cursor hotspot.
Returns
The created TuiCursor. TUI_NULL is returned if an error occurs.
Errors:
Possible errors in order are TUI_ERROR_NOT_INITIALIZED ,TUI_ERROR_NULL_IMAGE, TUI_ERROR_INVALID_CHANNEL_COUNT, and GLFW errors. The first error that occurs will cause the function to immediatly return.
Requirements:
This function must be called only while TUIC is initialized.
Pointer lifetime:
The returned TuiCursor must be destroyed before TUIC is terminated, using the function tuiCursorDestroy().
Thread safety:
This function must only be called on the same thread on which TUIC was initialized to ensure safe memory access.

Create a new TuiCursor from a 4 channel raw pixel array.

Parameters
pixel_widthThe width of the pixel array in pixels.
pixel_heightThe height of the pixel array in pixels.
raw_pixelsA pointer to the start of the pixel array. The pixel array must be ordered from left to right, top to bottom. Each pixel must be in the order RGBA.
hotspot_xThe x pixel coordinate of the cursor hotspot.
hotspot_yThe y pixel coordinate of the cursor hotspot.
Returns
The created TuiCursor. TUI_NULL is returned if an error occurs.
Errors:
Possible errors in order are TUI_ERROR_NOT_INITIALIZED, TUI_ERROR_NULL_PIXELS, TUI_ERROR_INVALID_PIXEL_DIMENSIONS, and GLFW errors. The first error that occurs will cause the function to immediatly return. Also, an inccorectly sized or allocated pixel array may cause undefined behaviour or a fatal crash without an error.
Requirements:
This function must be called only while TUIC is initialized.
Pointer lifetime:
The returned TuiCursor must be destroyed before TUIC is terminated, using the function tuiCursorDestroy().
Thread safety:
This function must only be called on the same thread on which TUIC was initialized to ensure safe memory access.
