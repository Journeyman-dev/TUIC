◆ tuiImageLoad()

TuiImage tuiImageLoad ( const char *  path,
int  expected_channel_count 
)
Parameters
pathThe path to the file.
expected_channel_countThe expected image channel count. If 0, none is expected.
Returns
The TuiImage. NULL is returned on error.
Errors
Throws TUI_ERROR_NULL_PATH if path is NULL. Throws TUI_ERROR_INVALID_CHANNEL_COUNT if expected_channel_count is not 0, 3, or 4. Throws TUI_ERROR_LOAD_IMAGE_FAILURE if image fails to load from the file. Throws TUI_ERROR_LOAD_IMAGE_UNEXPECTED_CHANNELS if expected_channel_count is not 0 and does not match the loaded image channel count.
