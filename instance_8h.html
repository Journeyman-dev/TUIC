◆ tuiInstanceDestroy()

void tuiInstanceDestroy ( TuiInstance  instance)

This function should be called when a TuiInstance object is no longer needed, typically before application exit or when replacing an existing instance with a new one that uses a different graphics backend. All opaque objects that use this instance need to be destroyed first, or otherwise a debug error message will be sent to the callback.

Parameters
instanceThe TuiInstance to destroy.
Errors
Throws TUI_ERROR_NULL_INSTANCE if instance is NULL. Throws TUI_ERROR_BACKEND_SPECIFIC and may or may not return if backend specific errors occur.
