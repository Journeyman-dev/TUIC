Enumerator
TUI_ERROR_UNKOWN 

TUI error for situations where an unkown error has occured.

TUI_ERROR_BACKEND_SPECIFIC 

TUI error for backend specific errors.

When this error is passed into a debug callback function, the 2nd argument is contains a textual description of the error. This is different from all other errors, which have the function name where the error was called passed into the 2nd argument.

TUI_ERROR_DANGLING_GLYPH_ATLAS 

TUI error for when a TuiInstance is destroyed when there are still one or more attached TuiGlyphAtlas objects.

TUI_ERROR_DANGLING_PALETTE 

TUI error for when a TuiInstance is destroyed when there are still one or more attached TuiPalette objects.

TUI_ERROR_DANGLING_PANEL 

TUI error for when a TuiInstance is destroyed when there are still one or more attached TuiPanel objects.

TUI_ERROR_DAMAGED_INSTANCE 

TUI error for when trying to use an opqaue object attached to a damaged TuiInstance or a damanged TuiInstance.

TUI_ERROR_INCOMPATIBLE_BLEND_MODE 

TUI error for when a TuiInstance is destroyed when there are still one or more attached TuiPanel objects.

TUI_ERROR_INCOMPATIBLE_IMAGES

msgFor errors with TuiErrorCode TUI_ERROR_BACKEND_SPECIFIC, this is a string description of the error. For all other error codes, this is the name of the function where the error occured.
